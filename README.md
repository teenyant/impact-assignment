# impact-assignment

## Problem

Implement code which has the ability to produce a comma delimited list of numbers, grouping the numbers into a range when they are sequential. 

## Requirements
- Java version 17
- Maven 3.9.9

## Build main

```bash
cd numberranges
mvn clean install
```

## Run unit tests
